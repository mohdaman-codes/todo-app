# ✅ Todo App

A clean and responsive Todo List application built using **HTML**, **CSS**, and **JavaScript**. Users can add, complete, filter, and delete tasks. The application stores data in **Local Storage**, ensuring tasks remain available even after refreshing the page.

---

## 🚀 Live Demo

https://todo-app-three-zeta-39.vercel.app/

---

## ✨ Features

- Add New Tasks
- Mark Tasks as Completed
- Delete Individual Tasks
- Clear Completed Tasks
- Filter Tasks
  - All
  - Active
  - Completed
- Remaining Items Counter
- Empty State Message
- Current Date Display
- Local Storage Support
- Fully Responsive Design
